docs.reactive-graph.io

On this page you will find the documentation for Reactive Graph. Reactive Graph is a reactive runtime based on a graph database, empowering everyone to build reliable and efficient software.

< Book >

The book covers the installation, configuration, and use of Reactive Graph.

It also describes the design principles, the data model, and the interfaces (GraphQL, Dynamic Graph, and the CLI).

Finally, the plugin system and plugin implementation are explained. Another chapter shows how to embed the runtime into your own application.
