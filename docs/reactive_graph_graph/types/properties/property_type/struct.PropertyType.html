pub struct PropertyType {

    pub extensions: Extensions,
}
Expand description

Definition of a property. The definition contains the name of the property, the data type and the socket type.

pub fn builder() -> PropertyTypeBuilder<((), (), (), (), (), ())>

Create a builder for building PropertyType. On the builder, call .name(...), .description(...)(optional), .data_type(...), .socket_type(...)(optional), .mutability(...)(optional), .extensions(...)(optional) to set the values of the fields. Finally, call .build() to create the instance of PropertyType.
